Title: Making my own website by using 'GitHub Pages'
Date: 2019-05-09

① Repository name: username.github.io
url: https://username.github.io (main page)
url: https://username.github.io/filename
url: https://username.github.io/filename.extension
· You can also access to
'https://username.github.io/username.github.io/'(main page)
'https://username.github.io/username.github.io/index'(main page)
'https://username.github.io/username.github.io/filename

② Repository name: projectname
url: https://username.github.io/projectname
url: https://username.github.io/projectname/filename
url: https://username.github.io/projectname/filename.extension

· Create 'index.html' file. This will be your main page.
· If your site is a User or Organization Page that has a repository named .github.io or .github.io,
you cannot publish your site's source files from different locations.
User and Organization Pages that have this type of repository name are only published from the master branch.
ref: https://help.github.com/en/articles/user-organization-and-project-pages
